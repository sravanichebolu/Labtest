胡椒鹽雜訊
主要利用rand產生0~1亂數，並假設當雜訊為10%時，則設定0.05以下及0.95以上時產生黑點或白點。

中值濾波法
除邊緣以外，將每點及其周圍8點存入一陣列，並經由insertion sort排列後回傳第5大值，取代原先的PIXEL。
而邊緣及角落沒有想到比較有效率的演算法，因此只能土法煉鋼將鏡射值存入陣列再做排序。

PSNR
因每一點均須計算產生雜訊前後均方差，但又因RGB的三個值所占比重不同，因此先依照R*0.299+G*0.587+B*0.114，將RGB轉為灰階後再進行計算。

補充:資料夾內有另外輸出的其他張圖片，那目前程式中用的是lena圖因為像素少一點可能跑得比較快。
